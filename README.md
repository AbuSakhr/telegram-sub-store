# 🛍️ بوت متجر الاشتراكات الرقمية (Telegram Sub Store)

بوت تلجرام متكامل لإدارة وبيع الاشتراكات الرقمية والحسابات بشكل تلقائي، مع لوحة تحكم كاملة للأدمن ونظام شحن رصيد يدوي.

## ✨ المميزات

- **نظام رصيد:** لكل مستخدم محفظة خاصة يتم الخصم منها عند الشراء.
- **شراء تلقائي:** تسليم فوري للحسابات بمجرد الضغط على زر الشراء.
- **لوحة تحكم الأدمن:** إدارة المنتجات، المخزون، المستخدمين، وطلبات الشحن من داخل التلجرام.
- **نظام المخزون:** إضافة حسابات متعددة لكل منتج ومنع التكرار.
- **شحن الرصيد:** نظام يدوي يتيح للمستخدم إرسال إثبات الدفع وموافقة الأدمن عليه.
- **إحصائيات شاملة:** متابعة المبيعات، الإيرادات، وأعداد المستخدمين.

## 🚀 التقنيات المستخدمة

- **Node.js** & **Telegraf** (Telegram Bot Framework)
- **SQLite** (قاعدة بيانات سريعة ومدمجة)
- **Winston** (نظام تسجيل السجلات/Logs)

## 🛠️ الإعداد والتشغيل

### 1. المتطلبات
- Node.js v18 أو أحدث.
- توكن بوت من [@BotFather](https://t.me/BotFather).
- معرف التلجرام الخاص بك (الأدمن).

### 2. التثبيت المحلي
```bash
# استنساخ المستودع
git clone https://github.com/AbuSakhr/telegram-sub-store.git
cd telegram-sub-store

# تثبيت المكتبات
npm install

# إعداد متغيرات البيئة
cp .env.example .env
# قم بتعديل ملف .env وإضافة التوكن ومعرف الأدمن
```

### 3. التشغيل
```bash
npm start
```

## 🌐 النشر على Railway

1. قم بربط مستودع GitHub الخاص بك بموقع [Railway.app](https://railway.app).
2. أضف متغيرات البيئة (Variables) في إعدادات المشروع:
   - `BOT_TOKEN`: توكن البوت.
   - `ADMIN_ID`: معرف التلجرام الخاص بك.
   - `PAYMENT_INFO`: معلومات الدفع الخاصة بك.
3. سيقوم Railway بالتعرف على `package.json` وتشغيل البوت تلقائياً.

## 🔧 أوامر البوت

- `/start` - بدء استخدام البوت.
- `/balance` - عرض رصيدك الحالي.
- `/admin` - فتح لوحة تحكم الأدمن (للأدمن فقط).
- `/help` - عرض دليل المساعدة.

## 📝 ملاحظات
- يتم تخزين البيانات في ملف `data/store.db`.
- تأكد من عدم مشاركة ملف `.env` أو ملف قاعدة البيانات علناً.
